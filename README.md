# 💰 Personal Expense Tracker Web Application

A full-stack web application to track daily expenses, manage budgets, and visualize spending using interactive dashboards and charts.

---

## 📌 Features

- 🔐 User Authentication (Register/Login)
- 💸 Add and manage expenses
- 📊 Dashboard with total expense summary
- 📈 Interactive charts (Pie & Bar)
- 🎯 Monthly budget tracking
- 🌙 Light/Dark mode toggle
- 🔔 Notifications and alerts
- ⚡ Smooth UI with animations

---

## 🛠️ Technologies Used

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** PHP
- **Database:** MySQL
- **Charts:** Chart.js
- **Server:** XAMPP

## ⚙️ Setup Instructions

1. Install XAMPP and start:
   - Apache
   - MySQL

2. Place project folder in: C:\xampp\htdocs\
3. Open phpMyAdmin: http://localhost/phpmyadmin
4. Create Database: expense_tracker
5. Write SQL or import pre-existing
6. Run Project: http://localhost/expense-tracker/frontend/index.html
